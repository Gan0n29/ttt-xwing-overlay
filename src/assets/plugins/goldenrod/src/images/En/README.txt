This folder contains all pilot and upgrade card images from X-Wing Second Edition.

Exported at 2020-11-24.

This was downloaded using https://github.com/guidokessels/xwing-image-downloader

All images have received their card XWD id as filename. For example: `Chancellor Palpatine` would be `chancellorpalpatine.png`
Dual cards will have the `-sideb` suffix appended to the XWS id. For example: `chancellorpalpatine-sideb.png`
